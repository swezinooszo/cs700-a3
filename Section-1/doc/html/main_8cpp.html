Detailed Description

This 'main.cpp' file is designed to sort arrays in three different orders: random, ascending, and reverse. It then calculates the time taken to perform each sorting operation. To analyze the impact of array size and initial data order, the program evaluates arrays of three different sizes: Hundred Thousand, One Million, and Ten Million elements. For each size, ten arrays are sorted in each of the three orders, and the execution times are stored in a two-dimensional array to present the final results. This file utilize the functions from "arrayUtil.h" file to achieve the desired result.
